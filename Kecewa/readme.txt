- Cara melakukan pertandingan antara Reference Bot dan Test Bot
Test Bot adalah bot hasil kreasi sendiri.
1.	Buka Command Prompt
2.	Masuk ke direktori “v1.2.6” hasil ekstraksi dengan Command Prompt
3.	Copy dan paste folder Test Bot (berisi bot.py, bot.json, dan file lainnya) ke direktori tersebut
4.	Masuk ke direktori “Game engine”
5.	Jalankan command Bomberman.exe --pretty -b “../Reference Bot” “../Test Bot”

- Cara menambahkan pemain dalam pertandingan
1.	Command yang dijalankan adalah Bomberman.exe --pretty -b “../Reference Bot” “../Test Bot” -cN dengan N adalah jumlah pemain yang ingin ditambahkan.
